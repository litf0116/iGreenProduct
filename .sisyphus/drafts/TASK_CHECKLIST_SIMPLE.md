## 六、技术说明

### APK打包方案

当前iGreenApp是Vite + React的H5项目，打包APK方案：

| 方案 | 工时 | 说明 |
|------|:----:|------|
| **Capacitor** | 4h | 推荐，与React生态兼容性好，可直接打包Android/iOS |
| **Cordova** | 4h | 成熟方案，插件丰富 |

**推荐使用 Capacitor**: 
```bash
npm install @capacitor/core @capacitor/android
npx cap add android
```
